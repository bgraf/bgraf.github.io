Projects

  • Course name generator (github)
    Set yourself apart from other candidates by solid education.

  • Rückblick (github)
    A travelogue static site generator.

    Features:

    • Markdown entries
    • Gallery support
    • GPX support through leaflet
    • Gallery images are mapped onto a GPX track by timestamp.
    • Calendar overview
    • Lazy generation / updates: only rewrite what has changed.

  • swplayer (github)
    Reimplementation of the sleep-well player.

    Scans directories for audio files and selects one or more randomly to play. After playing, shutdown is called. Played files are saved to a history file and this history influences the selection probabilities on future runs.

    I use this software daily to play short radio plays of Philip Maloney produced by the swiss SRF.
