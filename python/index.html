  • colorama
    Simple terminal output coloring.

  • pyperclip
    Read from and write to the clipboard.

  • polars
    Dataframes and data wrangling library.

    I prefer it over pandas because:

    • Columns are typed
    • The syntax is more intuitive and similar to tidyverse libraries (R)
    • A lot easier to read
    • Seems faster (maybe it is, maybe it isn’t)
    • Simple in- and output of parquet and excel files
    • Excel output comes per default with autofilters

  • PySCIPOpt
    Direct interface to SCIP, including quadratic programming.

    The API is not as nice as ortools’ API, but allows for more fine-grained control as it does not abstract over multiple solvers.

    Examples: https://github.com/mmghannam/scipdex/

  • ortools
    Optimization library for linear and integer programming. Some other heuristics as well.

    Why I like this library:

    • Nice interface
    • Comes bundled with the SCIP MIP-solver
    • Library and API also available in C++ and other languages
    • Allows for advanced features like lazy constraint callbacks and user cut callbacks, see example (1).
